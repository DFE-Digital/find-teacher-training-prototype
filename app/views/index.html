What you’ll need

GCSEs

To apply for a course you’ll need grade 4 (previously grade C) GCSEs in maths and English.

If you want to teach in a primary school you’ll also need a minimum grade 4 in a science subject.

If you don’t have these qualifications, you must prove you have the equivalent level of knowledge in these subjects.

Degree

You’ll need a UK degree, or the international equivalent.

Tests and checks

Before you start the course you’ll need to:

  • pass literacy and numeracy skills tests
  • have a Disclosure and Barring Service (DBS) security check
